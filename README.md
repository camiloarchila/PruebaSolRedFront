# Iniciando

## Prerequisitos

* [Angular] - framework de desarrollo web 
  
## Construccion

Asegurar que el codigo fuente del backend este corriendo 

Correr el proyecto utilizando 
```
ng serve 
```
y estara corriendo en 

```
http://localhost:8080/ 
```
